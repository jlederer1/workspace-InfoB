VIM COMMANDS
https://cognitivewaves.wordpress.com/vi-editor/  

hjkl                - navigation (left, down, up, right)
i|:|v| Esc (Caps)   - mode (insert, command, visual, normal)
    Shift+i             - insert from start of line 
    Shift+a             - insert from end of line
    i|a                 - insert left/right from cursor  

x|dd|u              - delete (del char/line) and undo
+y|+p|*y|*p               - copy from/ paste to local/ system clipboard

:set number         - enumerate lines
    :5                  - goto line 5

:w|:!       - save, run (e.g. :!javac hello.java ?)
